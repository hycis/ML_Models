

To compile, input into terminal

javac DecisionMaking.java KangJump.java Node.java

for PART 1

// output states for qn 1
java DecisionMaking p1q1

// output states for qn 2
java DecisionMaking p1q2

for PART 2

java KangJump $hole1 $hole2 $home $numOfCells

default for PART 2 is 
hole1 = 2
hole2 = 6
home = 4
numOfCells = 8

you can play with different settings to observe different results.
